How I learned programming

It all started when I was 9 or 8 years old. My parents knew I wanted to learn programming, so they bought me a book.

When I became 11, they bought me another book, entitled "Java from 0 to expert" by Stefan Tanasa, Stefan Andrei and Cristian Olaru. This book helped me when I had problems in java programming but I couldn't read it all because it is very long.

I didn't learn everything from books. The internet has helped me a lot. I didn't watch any particular YouTube channel to learn programming, but I know a site that taught me how to code in html, javascript and css: W3schools
